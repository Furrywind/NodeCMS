# NodeCMS

## Installation

```
# install dependencies
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```

## Usage

Change the `baseURL` to your backend address in [http.js](/src/api/http.js) before run.
